# NIC Startup Forge AI

Generative AI app for National Incubation Center - creates complete startup plans using Groq API.

## Features
- Black & Neon Blue Theme
- Landing Page + Startup Generator
- Groq Mixtral 8x7B integration
- Complete startup blueprint (MVP, Business Model, Roadmap)

pip install -r requirements.txt
streamlit run app.py
